# 💻 المشروع الثالث: واجهة ويب النحالين (Beekeeper Web Dashboard)
> **"مكتب النحال الرقمي"**

## 📍 الموقع الحالي
`frontend-web`

## 🎯 الهوية والهدف
نسخة مكتبية مريحة للنحالين وملاك المناحل الكبيرة لإدارة أعمالهم من الكمبيوتر. **ليست للمدراء**، بل هي "تطبيق الهاتف ولكن على شاشة كبيرة".

## 👥 المستخدمون
1. **مالك المنحل (Apiary Owner):** الإدارة المالية، التقارير التفصيلية، إدارة العمال.
2. **تاجر المستلزمات (Merchant):** (مستقبلاً) عرض المنتجات.
3. **النحال:** مراجعة سجلات طويلة لا يمكن عرضها في الهاتف بسهولة.

## 🛠️ الوظائف الأساسية
- **تمثيل كامل لتطبيق الهاتف:** نفس وظائف التطبيق (إدارة مناحل، أمراض) لكن بواجهة أوسع.
- **إدخال البيانات الكثيفة:** أسهل من الهاتف لإدخال بيانات 100 خلية دفعة واحدة.
- **الطباعة:** طباعة تقارير الفحص أو رموز QR للعاملين.
- **بيئة اختبار (Sandbox):** المكان الذي نختبر فيه الميزات الجديدة قبل وضعها في الهاتف.
