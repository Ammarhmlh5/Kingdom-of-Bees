## 🔐 آليات الأمان

### 1. Backend
```typescript
// Middleware للصلاحيات
- authenticate() - التحقق من Token
- requireAdmin() - صلاحيات الإدارة
- requireBeekeeper() - صلاحيات النحال
- requireOwnership() - التحقق من الملكية
```

### 2. Frontend
```typescript
// Guards للحماية
- RoleGuard - حماية صفحات النحالين
- AdminGuard - حماية صفحات الإدارة
```

### 3. Routes
```typescript
// فصل تام في الـ Routes

/dashboard   - واجهة النحالين

/hives       - واجهة النحالين
```
