## 📋 نظرة عامة

هذه المرحلة تركز على اختبار شامل لجميع المكونات المطورة في المراحل السابقة، وضمان جودة الكود، وتحسين الأداء، وتوثيق النظام.

## 🎯 الأهداف الرئيسية

1. ✅ **اختبار التكامل:** التأكد من عمل جميع المكونات معاً بسلاسة
2. ✅ **اختبار الأداء:** قياس وتحسين سرعة الاستجابة
3. ✅ **اختبار تجربة المستخدم:** التأكد من سهولة الاستخدام
4. ✅ **إصلاح الأخطاء:** معالجة جميع المشاكل المكتشفة
5. ✅ **توثيق الكود:** إضافة تعليقات وشروحات
6. ✅ **دليل المستخدم:** إنشاء دليل شامل للاستخدام

## 📝 قائمة الاختبارات

### 1. اختبار Backend APIs

#### 1.1 Inspection APIs
- [ ] **GET** `/api/apiaries/:apiaryId/hives/inspection-queue`
  - [ ] يرجع قائمة الخلايا مرتبة حسب الأولوية
  - [ ] يحسب الأولويات بشكل صحيح
  - [ ] يتعامل مع المنحل الفارغ
  - [ ] يتعامل مع منحل غير موجود (404)

- [ ] **POST** `/api/apiaries/:apiaryId/hives/:hiveId/inspect`
  - [ ] يسجل الفحص بنجاح
  - [ ] يحدث أولوية الخلية
  - [ ] يسجل في DailyOperations
  - [ ] يتعامل مع بيانات ناقصة (400)
  - [ ] يتعامل مع خلية غير موجودة (404)

#### 1.2 Split APIs
- [ ] **GET** `/api/apiaries/:apiaryId/hives/split-candidates`
  - [ ] يرجع الخلايا المرشحة للتقسيم
  - [ ] يحسب درجة القوة بشكل صحيح
  - [ ] يصنف الجاهزية (READY/SOON/NOT_READY)
  - [ ] يتعامل مع منحل بدون خلايا جاهزة

- [ ] **POST** `/api/apiaries/:apiaryId/hives/:hiveId/split`
  - [ ] ينفذ التقسيم بنجاح
  - [ ] ينشئ خلية جديدة
  - [ ] يسجل في SplitOperation
  - [ ] يسجل في DailyOperations
  - [ ] يتعامل مع بيانات غير صحيحة

#### 1.3 Merge APIs
- [ ] **GET** `/api/apiaries/:apiaryId/hives/merge-candidates`
  - [ ] يرجع الخلايا الضعيفة
  - [ ] يقترح عمليات دمج ذكية
  - [ ] يراعي الموسم (Spring/Autumn)
  - [ ] يحسب نسبة البقاء

- [ ] **POST** `/api/apiaries/:apiaryId/hives/:hiveId/merge`
  - [ ] ينفذ الدمج بنجاح
  - [ ] يحدث حالة الخلايا
  - [ ] يسجل في MergeOperation
  - [ ] يسجل في DailyOperations
  - [ ] يتعامل مع دمج خلية مع نفسها (400)

#### 1.4 Super APIs
- [ ] **GET** `/api/apiaries/:apiaryId/hives/super-candidates`
  - [ ] يرجع الخلايا الجاهزة للعاسلات
  - [ ] يعرض السياق الموسمي
  - [ ] يقترح الإطارات المناسبة
  - [ ] يحسب الجاهزية بشكل صحيح

- [ ] **POST** `/api/apiaries/:apiaryId/hives/:hiveId/super`
  - [ ] يضيف عاسلة بنجاح
  - [ ] يسجل في SuperOperation
  - [ ] يسجل في DailyOperations
  - [ ] يتعامل مع بيانات غير صحيحة

#### 1.5 Simulation APIs
- [ ] **POST** `/api/apiaries/:apiaryId/simulate`
  - [ ] يشغل المحاكاة لخلية واحدة
  - [ ] يشغل المحاكاة لعدة خلايا
  - [ ] يشغل المحاكاة للمنحل بالكامل
  - [ ] يحسب التوقعات لـ 12 شهر
  - [ ] يراعي العوامل المختلفة (طقس، موسم، نحال)
  - [ ] يحسب نسبة الثقة

#### 1.6 Operations APIs
- [ ] **GET** `/api/operations/daily`
  - [ ] يرجع العمليات اليومية
  - [ ] يفلتر حسب التاريخ
  - [ ] يفلتر حسب نوع العملية
  - [ ] يفلتر حسب العامل
  - [ ] يفلتر حسب المنحل
  - [ ] يرتب حسب التاريخ (الأحدث أولاً)

- [ ] **DELETE** `/api/operations/:operationId`
  - [ ] يحذف العملية بنجاح
  - [ ] يسجل عملية الحذف
  - [ ] يتعامل مع عملية غير موجودة (404)
