## 📊 البيانات التي ستظهر في لوحة التحكم

### 1. المؤشرات السريعة (Quick Stats Cards)

#### 🔢 إجمالي الخلايا
- **المصدر:** عدد جميع الخلايا في المنحل
- **الحقل:** `COUNT(hives)`
- **مثال:** إذا كان لديك 10 خلايا، سيظهر: **10**

#### ⭐ خلايا ممتازة
- **المصدر:** الخلايا ذات الحالة `EXCELLENT`
- **الحقل:** `condition = 'EXCELLENT'`
- **مثال:** إذا كان لديك 3 خلايا ممتازة، سيظهر: **3**

#### ⚠️ خلايا ضعيفة
- **المصدر:** الخلايا ذات الحالة `WEAK`
- **الحقل:** `condition = 'WEAK'`
- **مثال:** إذا كان لديك خلية ضعيفة واحدة، سيظهر: **1**

#### 🔔 تحتاج انتباه
- **المصدر:** الخلايا التي تحتاج فحص عاجل
- **الحساب:** الخلايا التي مر على آخر فحص لها أكثر من 14 يوم
- **الحقل:** `lastInspectionDate < (اليوم - 14 يوم)`
- **مثال:** إذا كان لديك 2 خلية لم تفحصها منذ أسبوعين، سيظهر: **2**

### 2. إحصائيات المنحل (Apiary Stats)

#### 📊 توزيع الحالة (Condition Distribution)

**الحقل المستخدم:** `condition` (من نوع `HiveCondition`)

**القيم الممكنة:**
- ✅ **EXCELLENT** (ممتازة): خلايا في أفضل حالة
- ✅ **VERY_GOOD** (جيدة جداً): خلايا قوية
- ✅ **GOOD** (جيدة): خلايا طبيعية
- ✅ **FAIR** (متوسطة): خلايا تحتاج مراقبة
- ✅ **WEAK** (ضعيفة): خلايا تحتاج تدخل

**مثال:**
```
ممتازة:    3 خلايا (30%)  ████████████░░░░░░░░░░░░░░░░░░░░
جيدة جداً: 2 خلايا (20%)  ████████░░░░░░░░░░░░░░░░░░░░░░░░
جيدة:      4 خلايا (40%)  ████████████████░░░░░░░░░░░░░░░░
متوسطة:    1 خلية  (10%)  ████░░░░░░░░░░░░░░░░░░░░░░░░░░░░
ضعيفة:     0 خلايا (0%)   ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░
```

#### 💪 توزيع القوة (Strength Distribution)

**الحقل المستخدم:** `strengthRating` (من نوع `StrengthRating`)

**القيم الممكنة:**
- 🟢 **قوية**: `VERY_STRONG` أو `STRONG`
- 🟡 **متوسطة**: `MEDIUM`
- 🔴 **ضعيفة**: `WEAK` أو `VERY_WEAK`

**مثال:**
```
┌─────────┬─────────┬─────────┐
│ قوية   │ متوسطة │ ضعيفة  │
│   5     │   3     │   2     │
└─────────┴─────────┴─────────┘
```

#### 📐 متوسط الإطارات

**الحساب:** 
```
متوسط الإطارات = مجموع عدد الإطارات لكل خلية ÷ عدد الخلايا
```

**المصدر:** 
- عدد الإطارات من جدول `HiveFrame`
- `COUNT(frames) WHERE hiveId = x`

**مثال:**
- خلية 1: 10 إطارات
- خلية 2: 12 إطار
- خلية 3: 8 إطارات
- **المتوسط:** (10 + 12 + 8) ÷ 3 = **10 إطارات**

#### 👑 أعمار الملكات

**الحقل المستخدم:** `queenAge` (بالأشهر)

**التصنيف:**
- 🟢 **صغيرة**: ≤ 12 شهر (سنة واحدة أو أقل)
- 🟡 **متوسطة**: 13-24 شهر (1-2 سنة)
- 🔴 **كبيرة**: > 24 شهر (أكثر من سنتين)

**مثال:**
```
┌──────────────────────────────────┐
│ صغيرة: 4  │ متوسطة: 3 │ كبيرة: 2 │
└──────────────────────────────────┘
```

### 3. لوحة المحاكاة (Simulation Panel)

**البيانات:**
- 📈 **توقعات 12 شهر قادم** لكل خلية
- 📊 **رسم بياني** يوضح التطور المتوقع
- 🎯 **توصيات** بناءً على التوقعات

**المصدر:** 
- يتم حسابها بناءً على:
  - حالة الخلية الحالية
  - تاريخ الفحوصات
  - الموسم الحالي
  - معدل النمو

**مثال:**
```
الشهر 1: قوة 75% - حالة جيدة
الشهر 2: قوة 80% - حالة جيدة جداً
الشهر 3: قوة 85% - حالة ممتازة
...
```

### 4. لوحة التنبيهات (Alerts Panel)

**أنواع التنبيهات:**

#### 🔴 أولوية عالية (High Priority)
- خلايا ضعيفة تحتاج تدخل فوري
- خلايا لم تُفحص منذ أكثر من 21 يوم
- خلايا بدون ملكة

#### 🟡 أولوية متوسطة (Medium Priority)
- خلايا تحتاج فحص (14-21 يوم)
- خلايا بملكة كبيرة (> 2 سنة)
- خلايا متوسطة الحالة

#### 🟢 أولوية منخفضة (Low Priority)
- توصيات عامة
- تذكيرات صيانة

**مثال:**
```
┌─────────────────────────────────────┐
│ 🔴 خلية A-01 تحتاج فحص عاجل      │
│    آخر فحص: منذ 25 يوم             │
├─────────────────────────────────────┤

│    عمر الملكة: 28 شهر              │
├─────────────────────────────────────┤
│ 🟢 خلية C-05 جاهزة للتقسيم       │
```

## 🗄️ الحقول المستخدمة من قاعدة البيانات

### من جدول `Hive`:

| الحقل | النوع | الاستخدام |
|-------|-------|-----------|
| `id` | String | معرّف الخلية |
| `hiveNumber` | String | رقم الخلية |
| `name` | String? | اسم الخلية |
| `condition` | HiveCondition | حالة الخلية (ممتازة، جيدة، ضعيفة...) |
| `strengthRating` | StrengthRating? | تقييم القوة (قوية، متوسطة، ضعيفة) |
| `strengthScore` | Int? | نقاط القوة (0-100) |
| `queenAge` | Int? | عمر الملكة بالأشهر |
| `lastInspectionDate` | DateTime? | تاريخ آخر فحص |
| `nextInspectionDue` | DateTime? | موعد الفحص القادم |
| `status` | HiveStatus | حالة النشاط (ACTIVE, INACTIVE...) |

### من جدول `HiveFrame`:

| الحقل | النوع | الاستخدام |
|-------|-------|-----------|
| `hiveId` | String | معرّف الخلية |
| `position` | Int | موضع الإطار |
| `frameType` | FrameType | نوع الإطار |
| `broodPercentage` | Int | نسبة الحضنة |
| `honeyPercentage` | Int | نسبة العسل |

### من جدول `Inspection`:

| الحقل | النوع | الاستخدام |
|-------|-------|-----------|
| `hiveId` | String | معرّف الخلية |
| `inspectionDate` | DateTime | تاريخ الفحص |
| `queenSeen` | Boolean | هل شوهدت الملكة |
| `broodFrames` | Int | عدد إطارات الحضنة |
